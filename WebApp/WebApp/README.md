# Online Library

# Configuración del proyecto.

- Para configurar la libreria se deben correr el migration, con el comando add-migration initial y luego update-database.
- Luego de tener la base de datos lista, se deben correr los scripts que estan en el file Library.sql en la base de datos para 
agregar un libro y 2 estados a la entidad estados.

# Get started

Simplemente se debe correr la aplicación y aparecera en la pantalla de inicio donde se veran los libros disponibles, 
a la derecha debajo de la imagen se puede logear o crear un usuario para asi poder ver las paginas del libro creado.
